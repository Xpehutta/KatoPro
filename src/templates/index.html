2. Параметры генерации

Укажите месяц и точки. Если точки не выбраны — будут обработаны все из списка. Если в data/ нет файла предыдущего периода — сервис предложит скорректировать период по каждой точке. Если файл за этот период уже есть в generated/, приложение спросит подтверждение перед пересозданием.

Как добавить точку

  • Пакетная загрузка — выберите сразу несколько `.xlsx`;
  • Импорт из data/ — зарегистрировать все файлы, уже лежащие в папке;
  • Файлы на диске — смотрите, скачивайте и удаляйте исходники и результаты;
  • при удалении файл сначала подтверждается и уходит в папку trash/;
  • имя можно взять из файла: МесОтч202603_Смола.xlsx → «Смола».

Что заполняется автоматически

  • даты с 1-го по последнее число месяца;
  • подсветка выходных и праздников РФ;
  • формулы выручки, остатка, сумм и средних;
  • ссылки на остаток и ТМ предыдущего месяца;
  • защита листа с открытыми ячейками для ввода.

Полезные подсказки

  • Генерируйте месяцы по порядку — так корректнее связки с предыдущим листом.
  • Список точек хранится в data/points.yaml.
  • Можно добавить сколько угодно точек — не только 13.
  • Если API праздников недоступен, используются основные федеральные праздники.
