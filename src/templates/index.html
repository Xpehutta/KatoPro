Как добавить точку

  • Загрузка — выберите .xlsx с историей месяцев;
  • Вручную — если файл уже лежит в data/;
  • имя можно взять из файла: МесОтч202603_Смола.xlsx → «Смола».

Что заполняется автоматически

  • даты с 1-го по последнее число месяца;
  • подсветка выходных и праздников РФ;
  • формулы выручки, остатка, сумм и средних;
  • ссылки на остаток и ТМ предыдущего месяца;
  • защита листа с открытыми ячейками для ввода.

Полезные подсказки

  • Генерируйте месяцы по порядку — так корректнее связки с предыдущим листом.
  • Список точек хранится в data/points.yaml.
  • Можно добавить сколько угодно точек — не только 13.
  • Если API праздников недоступен, используются основные федеральные праздники.
